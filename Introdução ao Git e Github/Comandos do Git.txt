Comandos usados no Desafio
git clone (para clonar repositório do github)
ctrl + L (limpa a tela do Git)
git status (mostra status, se tem pasta nova ou editada)
git add . (adiciona conteudos)
git commit -m (adiciona-se um comentário "entre as aspas" sobre a última alteração do arquivo)
git push origin main (empurra o arquivo atualizado com os ultimos commits para o github)
